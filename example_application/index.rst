Example Application
===================

This section of the documentation will step you through the skeleton application to demonstrate
the various concepts explained the previous sections. The example application will step you through
a basic MVC application with some simple CRUD functionality using a small SQLite database. Contained
within the application is a demonstration of both a web-based and cli-based application.
