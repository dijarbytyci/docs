Pop\\Feed
=========

The `popphp/pop-feed` component manages the creation and parsing of standard web feeds.

Installation
------------

Install it directly into your project:

.. code-block:: bash

    composer require popphp/pop-feed

Or, include it in your composer.json file:

.. code-block:: json

    {
        "require": {
            "popphp/pop-feed": "2.0.*",
        }
    }
